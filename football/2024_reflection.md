# 2024年踢球技巧

## 2024-07-03

总结：

1. 耐心控球，不确定情况时先控住球
2. 过人时敢于做假动作
3. 不了解身后情况切勿贸然转身
4. 带球时要及时触球、小力度控球
5. 往人少、有空间的地方带球或跑位

## 2024-06-19

总结：

1. 要一直提前观察，不能注意力全在球上
2. 多人包抄时要么传球，要么短距离控球，比如使用油炸丸子
3. 回追要控制速度和重心
4. 主动过人时控制好距离
